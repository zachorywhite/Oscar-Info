Our website pulls Oscar nominated movies from our database and displays them on the page. Each movie contains an ID value, the year it was nominated, the category in which it was nominated, whether it won or not, and who was nominated. The webpage contains 5 different ways to search for movies:

  • Pressing search button without selections displays every nomination
  • Only first dropdown year selected displays every nomination from that year
  • Only first dropdown year selected and category from that year displays nominations of selected category from the selected year
  • First year and second year dropdowns only selected displays range of nominations between selected years
  • First year, second year, and category dropdown selected displays nominations of a shared category from between the range of the two selected years

    • The data is queried from our database by our API to provide REST endpoints that deliver resources in JSON.
    By using the HTTP method GET on our REST API we are able to return collections/singleton resources and access them.

    Read Entire List:
    GET read.php 
    
http://oscarinfo-com.stackstaging.com/api/movie/read.php
    Using the GET method on the read.php page will provide a collection of all the Oscar nominations as an array of objects.

    Read One Category:
    GET read_category.php 
    
http://oscarinfo-com.stackstaging.com/api/movie/read_category.php?category={category}
    Using the GET method on the read_category.php will return all Oscar nominees of the selected category (category joined by +).

    EXAMPLE: category = directing+(comedy+picture) 
    
[
    {
        "id": "12",
        "year": "1927",
        "category": "DIRECTING (Comedy Picture)",
        "winner": "TRUE",
        "entity": "Lewis Milestone"
    },
    {
        "id": "13",
        "year": "1927",
        "category": "DIRECTING (Comedy Picture)",
        "winner": "FALSE",
        "entity": "Ted Wilde"
    }
]

    Read Multiple Years:
    GET read_multiyear.php 
    
http://oscarinfo-com.stackstaging.com/api/movie/read_multiyear.php?fyear={year}&lyear={year}
    Using the GET method on read_multiyearCat.php returns all Oscar nominees of a shared category (category joined by +) between the range of two selected years(fyear & lyear).

    Read Single Movie:
    GET read_single.php 
    
http://oscarinfo-com.stackstaging.com/api/movie/read_single.php?id={id}
    Using the GET method on read_single.php returns a singleton resource of one single movie of the given ID number (id).

    EXAMPLE: id=3157 
    
{
        "id": "3157",
        "year": "1955",
        "category": "ACTOR",
        "winner": "TRUE",
        "entity": "Ernest Borgnine"
 }

    Read Single Year:
    GET read_year.php 
    
http://oscarinfo-com.stackstaging.com/api/movie/read_year.php?year={year}
    Using the GET method on read_year.php returns all Oscar nominees of a selected year (year).

    Read One Year with Category:
    GET read_yearcat.php 
    
http://oscarinfo-com.stackstaging.com/api/movie/read_yearcat.php?year={year}&category={category}
    Using the GET method of read_yearcat.php returns all Oscar nominees of a selected category (category joined by +) from a single year (year)

    EXAMPLE: year=1927 & category=actor
